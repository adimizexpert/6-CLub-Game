# Setup Guide for 6Club Guide Website

## Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Add Images**
   Place the following images in `public/images/`:
   - `logo.png` (recommended: 240x80px)
   - `6-club-main.webp` (recommended: 1200x600px)
   - `K3-Lotre.webp`
   - `TRXhash.webp`
   - `5d.webp`
   - `6clubcolour.webp`
   - `WingoGame.webp`
   - `ios-appstore.webp`
   - `android-play.webp`
   - `about.webp`

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   npm start
   ```
