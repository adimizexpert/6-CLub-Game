# Images Required for 6Club Guide Website

Place all images in this directory (`public/images/`).

## Required Images

### Main Images
1. **logo.png** - Logo for the navbar (recommended: 240x80px or similar aspect ratio)
   - Location: `public/logo.png` (root of public folder)
   - Used in: Navbar component

2. **6-club-main.webp** - Main hero image on homepage (recommended: 1200x600px)
   - Location: `public/images/6-club-main.webp`
   - Used in: Home page hero section

### Game Images
3. **K3-Lotre.webp** - K3 Lotre game image (recommended: 800x400px)
   - Location: `public/images/K3-Lotre.webp`
   - Used in: Home page games section, K3 Lotre page

4. **TRXhash.webp** - TRX Hash game image (recommended: 800x400px)
   - Location: `public/images/TRXhash.webp`
   - Used in: Home page games section, TRX Hash game page

5. **5d.webp** - 5D Lotre game image (recommended: 800x400px)
   - Location: `public/images/5d.webp`
   - Used in: Home page games section, 5D Lotre page

6. **6clubcolour.webp** - Colour Prediction game image (recommended: 800x400px)
   - Location: `public/images/6clubcolour.webp`
   - Used in: Home page games section, Colour Prediction page
   - Note: if not provided, the site falls back to `public/images/6clubcolour.svg`

7. **WingoGame.webp** - Wingo game image (recommended: 800x400px)
   - Location: `public/images/WingoGame.webp`
   - Used in: Home page games section, Wingo game page

## Image Optimization Tips

- Use WebP format for main images when possible (better compression)
- PNG format for logos and images with transparency
- Optimize images before uploading (use tools like TinyPNG, ImageOptim, or Squoosh)
- Ensure images are properly sized (don't use oversized images)
- Use descriptive alt text (already implemented in code)
